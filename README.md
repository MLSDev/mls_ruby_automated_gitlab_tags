## Installation
Add this line to your application's Gemfile:

```ruby
gem 'mls_ruby_automated_gitlab_tags'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install mls_ruby_automated_gitlab_tags
```
